# C Programming: Variables and Data Types Exercises

Welcome to the **C Programming Variables and Data Types Exercises** repository! This collection of exercises is designed to help you master the fundamentals of variables, data types, and basic input/output (I/O) in C programming. Whether you're a beginner or looking to refresh your skills, these exercises will provide valuable practice.

## What You'll Learn

Through these exercises, you will:

- Understand variable declaration, initialization, and assignment.
- Work with different data types such as `int`, `float`, `double`, and `char`.
- Perform arithmetic operations, including modulo and type casting.
- Explore advanced basics like the `sizeof` operator and octal/hexadecimal literals.

Each exercise includes:
- **Problem Statement**
- **Solution Code**
