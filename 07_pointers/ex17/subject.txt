Exercise 17: Using calloc

Practice Problem: Use calloc() to dynamically allocate memory for an array of 10 floating-point numbers. Print the first and last elements to verify that calloc automatically initializes all elements to zero. Then, free the memory.

Given:

int count = 10;

Expected Output:

Verifying calloc initialization (should be 0.00):
First element (float_arr[0]): 0.00
Last element (*(float_arr + 9)): 0.00
Element at index 5 after assignment: 3.14
