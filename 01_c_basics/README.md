# Ultimate C Programming Exercises for Beginners

Welcome to the **Ultimate C Programming Exercises** repository! This collection contains **48 beginner-friendly exercises** designed to help you master the fundamentals of C programming. Each exercise focuses on a specific concept, allowing you to build your skills step by step.

---

## 📚 Topics Covered

These exercises cover a wide range of topics, including:

- **Fundamentals**: Input/Output, variables, arithmetic operations, type casting.
- **Conditionals and Loops**: `if-else`, `switch`, `for`, `while`, `do-while`.
- **Data Structures**: Arrays, strings, structures (nested and arrays of structures).
- **Functions**: Definition, calling, return types, recursion.
- **Pointers**: Declaration, dereferencing, pass-by-reference, pointer arithmetic.

---

## 📝 How to Use This Repository

1. Navigate to the folder for the exercise you want to work on (e.g., `ex01`, `ex02`, etc.).
2. Read the problem statement in the `subject.txt` file.
3. Implement your solution in the `solution.c` file.
4. Compile and run your code to test your solution.
