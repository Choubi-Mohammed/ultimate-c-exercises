Exercise 16: Check Array Palindrome
Practice Problem: Create a C program that determines whether an array of integers is a palindrome. An array is a palindrome if it reads the same forwards and backwards.

Given:

int arr1[] = {1, 2, 3, 2, 1}; // Palindrome
int arr2[] = {1, 2, 3, 4, 5}; // Not a palindrome
Expected Output:

Array 1 is a palindrome.
Array 2 is NOT a palindrome.
