Exercise 12: Frequency Count
Practice Problem: Write a C program to count how many times a particular element (provided by the user or hardcoded) appears in a given array.

Given:

int arr[] = {1, 5, 2, 8, 5, 1, 5, 9};
int search_element = 5;
Expected Output:

Element 5 appears 3 times in the array.
